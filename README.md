## 一、课程内容

MEMS技术是近年来随着微电子技术的不断成熟而发展起来的一门新兴的交叉学科。
其涵盖领域包括光学、机械、电子、计算机等诸多门类。
随着MEMS技术发展及应用的扩展，对其微细结构的性能及结构的测试要求也日益迫切。
由于其结构尺度属微纳米量级，因此传统意义的测试技术，几乎无能为力。
本课程针对MEMS装配领域中的目标物识别，通过图象处理处理手段，利用计算机技术及CCD图像采集，进行微型圆孔的自动图像识别。
